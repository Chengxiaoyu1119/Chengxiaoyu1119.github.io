# Chengxiaoyu1119.github.io

一个部署在 GitHub Pages 的个人博客静态站点仓库。

站点地址：<https://chengxiaoyu1119.github.io/>

## 技术栈

- Hexo 生成的静态页面
- Fluid 主题相关前端资源（HTML/CSS/JS）

## 维护说明

本仓库以静态产物为主。博客内容更新后，重新生成并同步静态文件到本仓库即可。

## 关键目录

- `about/`：关于页面
- `archives/`：归档页面
- `categories/`：分类页面
- `tags/`：标签页面
- `js/`：前端脚本
- `css/`：样式资源
- `img/`：图片资源
